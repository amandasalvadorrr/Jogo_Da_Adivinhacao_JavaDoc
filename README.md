## Objetivos

- Implementar um jogo simples em Java usando classes  
- Separar as responsabilidades em classes  
- Documentar o código utilizando **Javadoc**:
  - Classes  
  - Atributos  
  - Métodos  
  - Parâmetros e retornos  
  - Exceções (se houver)  
- Gerar documentação HTML  
- Analisar a importância da documentação mesmo em projetos simples  

---

## Descrição do jogo

O jogo **"Adivinha o Número"** funciona assim:

- O computador escolhe um número aleatório entre **0 e 100**  
- O jogador tenta adivinhar qual é o número  
- A cada tentativa, o jogo informa se o palpite é **maior**, **menor** ou **igual** ao número secreto  
  - Também pode usar mensagens como **"quente"**, **"morno"** ou **"frio"**  
- O jogo conta quantas tentativas o jogador fez  
- Quando o jogador acerta, o jogo exibe uma mensagem de parabéns e o número total de tentativas
